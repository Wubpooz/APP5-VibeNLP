## Informations générales

- **Date** : 31 mars 2026 (mardi)
- **Horaire** : 13h30 - 17h45
- **Salle** : 620-B
- **Format** : Présentations de projets en groupe + Notation par investissement

## Déroulement

### Phase 1 : Présentations des projets

- Chaque groupe présente son projet NLP à tour de rôle
- Toutes les présentations doivent être terminées avant la phase de notation

### Phase 2 : Notation par investissement

1. Chaque groupe reçoit un montant identique de **fonds d'investissement virtuels** (par ex. : 1 000 000)
2. Chaque groupe doit répartir la totalité de ses fonds entre les **autres groupes** (il est interdit d'investir dans son propre groupe)
3. La répartition est entièrement libre — concentration ou diversification au choix
4. Les investissements sont soumis de manière **anonyme**
5. Tous les groupes soumettent leur répartition simultanément

### Phase 3 : Résultats et notation

- Le total des investissements reçus par chaque groupe est comptabilisé
- Un classement est établi selon les montants obtenus
- **La note finale est déterminée par l'enseignant sur la base des résultats d'investissement**

## Règles

| Règle | Détails |
|-------|---------|
| Pas d'auto-investissement | Chaque groupe ne peut investir que dans les autres groupes |
| Répartition totale obligatoire | L'intégralité des fonds doit être distribuée, sans en retenir |
| Soumission anonyme | Les plans d'investissement sont soumis anonymement pour éviter la pression sociale |
| Présenter d'abord, investir ensuite | Toutes les présentations doivent être terminées avant le tour d'investissement |
| Appréciation de l'enseignant | L'enseignant se réserve le droit d'ajuster les notes finales selon son évaluation globale |
