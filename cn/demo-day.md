## 基本信息

- **日期**：2026年3月31日（周二）
- **时间**：13:30 - 17:45
- **地点**：620-B
- **形式**：小组项目展示 + 投资评分

## 流程

### 第一阶段：项目展示

- 各小组轮流上台展示自己的NLP项目
- 所有展示结束后，进入投资评分环节

### 第二阶段：投资评分

1. 每组获得相同数额的**虚拟投资基金**（例如：100万）
2. 每组必须将全部基金分配给**其他小组**（不能投资自己）
3. 分配完全自由——可以集中投资，也可以分散投资
4. 投资以**匿名**方式提交
5. 所有组同时提交投资分配方案

### 第三阶段：结果与评分

- 统计每组获得的总投资额
- 根据投资排名生成排行榜
- **最终分数由老师综合投资结果决定**

## 规则

| 规则 | 说明 |
|------|------|
| 不能投自己 | 每组只能将基金分配给其他小组 |
| 必须全部分配 | 基金总额必须全部投出，不能保留 |
| 匿名投资 | 投资方案匿名提交，避免社交压力 |
| 先展示后投资 | 所有展示全部结束后，统一进行投资评分 |
| 老师最终裁量 | 老师保留根据实际情况调整最终分数的权利 |
