- **主题数量**：4个，难度递增

## 主题说明

以下内容为参考方向，不是必须逐项覆盖的清单。每组根据自己的理解选择重点，讲清楚、讲透比面面俱到更重要。

### 主题 #1：创业工具箱（09/03）

为 Demo Day 的 NLP 项目做准备，分享实用工具和经验：

- 项目管理（如 Linear、GitHub Projects）
- 免费 LLM API 获取
- Vibe Coding 工具
- 前端部署（如 Vercel）
- 后端托管

### 主题 #2：Claude Code 深度使用（13/03）

掌握 Claude Code 的高级功能和工作流：

- MCP（Model Context Protocol）
- Skills 创建与使用
- Prompt 工程与 CLAUDE.md
- Tool Use 技巧
- 终端环境（如 Conductor、tmux）

### 主题 #3：Attention Is All You Need（19/03）

理解 Transformer 架构的核心：

- Self-Attention 机制（Query、Key、Value）
- Multi-Head Attention
- Positional Encoding
- Encoder-Decoder 架构
- 从 Transformer 到 BERT、GPT

### 主题 #4：OpenClaw 与 Agent 生态（20/03）

理解 AI Agent 的设计与发展趋势：

- OpenClaw 配置与使用
- OpenClaw 的设计理念与架构
- 平台生态（如 Moltbook）
- Agent-to-Agent（A2A）协议与趋势

## 评分标准

这不是传统的学术报告，而是一次**知识分享与学习反思**。

| 维度 | 说明 | 占比 |
|------|------|------|
| 知识传递 | 同学听完后是否真正学到了东西 | 40% |
| 学习过程与方法 | 是否展示了如何学会这个主题（用了什么AI工具、和同学怎么讨论的、遇到什么困难） | 40% |
| 展示质量 | 表达清晰、结构合理、时间控制在20分钟内 | 20% |
